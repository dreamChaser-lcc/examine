# 记录一些开发中遇到的问题

## ::before 和 ::after伪元素
:before和::before的区别,::是新规范,为了区分伪元素和:hover之类选择器的方法,IE8只支持单冒号的伪元素
```less
// 错误例子: 选择器&与伪元素之间,千万不能有空格,否则,子标签都会出现before
  .block{
    & ::before{
      content:'before content'
    }
    & ::after{
      content:'after content'
    }
  }
// 正确例子
  .block{
    &::before{
      content:'before content'
    }
    &::after{
      content:'after content'
    }
  }
```
